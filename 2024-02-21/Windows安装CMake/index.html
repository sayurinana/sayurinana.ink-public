Windows下安装CMake

本文环境：

  • OS：Windows 10
  • CMake 3.29.0-rc1

使用安装程序安装

官网链接：Download CMake

或见资源列表中的CMake 3.29.0-rc1 安装程序，若使用分享的安装包则可以直接跳转至安装部分

获取CMake安装程序

访问官网链接，单击Windows x64 Installer的文件链接开始下载，

使用CMake安装程序安装

双击运行安装程序，直到安装选项，

建议选上Add CMake to the system PATH，可以省去手动添加环境变量的步骤，

all users 表示添加到系统变量，current user 表示添加到当前用户的用户变量，

然后一直Next到安装就行，
